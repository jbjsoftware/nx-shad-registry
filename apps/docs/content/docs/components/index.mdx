## Features

All form controls in this library include:

- **React Hook Form Integration**: Seamless integration with form state management
- **Validation Support**: Built-in error handling and validation states
- **Accessibility**: WCAG compliant with proper ARIA attributes
- **TypeScript Support**: Full type safety and IntelliSense
- **Customizable**: Flexible styling and configuration options
